## Tech Debt

**Phase 15 (2 items — non-blocking):**

1. **OEMBED-03 viewer JS — no unit tests.** JavaScript is embedded inline in a Python string constant in `static_export.py`. No JS test infrastructure exists in this project. Verified manually via Netlify deploy. Accepted: scope is intentionally limited to unit-testable Python code.

2. **twttr.widgets.load() on re-render is best-effort.** Twitter widget SDK auto-processes `.twitter-tweet` blockquotes on initial `widget.js` load. Re-renders triggered by search/filter navigation may not re-process newly visible embeds until SDK is fully loaded. This is standard Twitter widget SDK behavior — widgets still appear correctly after a brief CDN load delay. Not a defect.
