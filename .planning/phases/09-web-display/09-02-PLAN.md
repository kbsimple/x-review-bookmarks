   - Show "Reposted from @{{ embedded_post.author_username }}" in text-sm text-gray-500

   - Render nested card with bg-gray-50 border border-gray-200 rounded p-3
   - If embedded_post.available: render embedded content inside nested card
   - If NOT embedded_post.available: call render_unavailable_placeholder(embedded_post)
   - "View on X" link points to the quote tweet (post.x_post_id)

5. Macro `render_unavailable_placeholder(embedded_post)` per D-05, D-06:
   - Div with bg-gray-100 rounded p-4 text-center
   - SVG icon (generic post icon from UI-SPEC.md)
   - Text: "Original post unavailable"
   - If embedded_post.author_username exists: show "@{{ embedded_post.author_username }}" in text-xs text-gray-400

   - This prevents XSS from malicious content in posts/embedded posts

1. Create src/web/templates/components/_lightbox.html with modal structure:
   ```html
   {% macro render_lightbox_script() %}
   <script>
   // Simple lightbox: click image to expand, click outside to close
   function openLightbox(imgSrc, imgAlt) {
       const lightbox = document.getElementById('image-lightbox');
       const img = document.getElementById('lightbox-img');
       img.src = imgSrc;
       img.alt = imgAlt;
       lightbox.classList.remove('hidden');
   }
   function closeLightbox() {
       const lightbox = document.getElementById('image-lightbox');
       lightbox.classList.add('hidden');
   }
   // Close on escape key
   document.addEventListener('keydown', (e) => {
       if (e.key === 'Escape') closeLightbox();
   });
   </script>
   {% endmacro %}

   {% macro render_lightbox_modal() %}
   <div id="image-lightbox" class="fixed inset-0 bg-black bg-opacity-75 hidden z-50 flex items-center justify-center" onclick="closeLightbox()">

       <img id="lightbox-img" class="max-w-full max-h-full object-contain" onclick="event.stopPropagation()">
   </div>
   ```
