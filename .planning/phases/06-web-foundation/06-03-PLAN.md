# PLAN: Post Browsing

**Phase:** 6 (Web Foundation)
**Wave:** 3 of 4
**Plan:** 06-03
**Depends on:** 06-02 (Web Authentication)
**Status:** ✅ Complete

## Goal

Implement post browsing with cursor-based pagination and post card rendering.

## Requirements

- WEB-04: User can browse posts with cursor-based pagination

## Success Criteria

1. Home page displays posts in reverse chronological order
2. Cursor-based pagination returns next page of posts
3. "Load more" button fetches next page via HTMX
4. Post cards show author, text preview, topics, and date
5. Clicking a post shows full content in modal or detail view

## Implementation Plan

### Step 1: Create Browse Routes

`src/web/routes/browse.py`:
- `GET /` — home page with initial posts
- `GET /browse` — browse page with filters
- `GET /api/posts` — JSON endpoint for infinite scroll (HTMX)

### Step 2: Implement Cursor Pagination

`src/web/pagination.py`:
- Cursor = `{created_at}:{x_post_id}` for stable ordering
- Decode cursor to get pagination parameters
- Use `PostsRepository.get_all_ordered(order="newest", limit=N, after_cursor=...)`

### Step 3: Add Pagination to PostsRepository

`src/repositories/posts.py`:
- Add `get_all_paginated(cursor, limit)` method
- Return posts plus `next_cursor` for next page
- Handle edge cases: no cursor (first page), no more posts (empty cursor)

### Step 4: Create Post Card Template

`src/web/templates/components/post_card.html`:
- Author avatar and username
- Text preview (truncated to 200 chars)
- Topic tags (styled chips)
- Publication date
- Link to original X post

### Step 5: Create Home Page Template

`src/web/templates/home.html`:
- Extends `base.html`
- Hero section with app name and description
- Post grid (responsive: 1 column mobile, 2 columns tablet, 3 columns desktop)
- "Load more" button with HTMX attributes

### Step 6: HTMX Infinite Scroll

`src/web/static/js/app.js`:
- HTMX handles "Load more" button
- `hx-get="/api/posts?cursor=${next_cursor}"`
- `hx-swap="beforeend"` to append posts
- Alternatively: use HTMX `hx-trigger="revealed"` for auto-scroll loading

### Step 7: Post Detail Modal

`src/web/templates/components/post_modal.html`:
- Full text content
- All metadata: author, date, topics, notes
- Links: original X post, copy text
- Close button (X or click outside)
