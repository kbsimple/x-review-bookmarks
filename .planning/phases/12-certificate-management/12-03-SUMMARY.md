## Platform Guide Content

### macOS
- `mkcert -install` installs CA in Keychain automatically
- Safari and Chrome trust certificates immediately
- CA certificate path shown for mobile device transfer

### Windows
- `mkcert -install` installs CA in Current User store
- Edge and Chrome trust certificates immediately
- PowerShell command for Local Machine installation

### Linux
- `mkcert -install` adds CA to system trust store
- Firefox requires NSS tools (`libnss3-tools`)
- `sudo update-ca-certificates` for Debian/Ubuntu

### iOS (per D-08)
- AirDrop/email/web download for certificate transfer
- Profile installation via Settings
- **CRITICAL**: Certificate Trust Settings must be toggled ON
- Red highlight for the trust step

### Android
- USB/cloud/email for certificate transfer
- Settings > Security > Install certificate
- Chrome works with user-installed CA
- Network security config for custom apps
