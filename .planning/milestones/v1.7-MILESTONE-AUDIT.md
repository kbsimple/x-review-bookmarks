### Phase 16 / Phase 17 Coexistence

- `renderCardFooter()` change is backward-compatible — all 4 card renderers (original, retweet, quote, oEmbed) call it; "View on X" link preserved inside the function

### Behavioral Edge Case (observation, not blocker)

Pressing Escape (which calls `setMode('stream')`) while in a deep-link carousel would switch to stream mode content but keep `XBM Home` button visible, since `deep-link-mode` class is not cleared by `setMode()`. The mode-switcher buttons are CSS-hidden in deep-link mode so this is only reachable via keyboard. Informs potential Phase 18 cleanup if needed.
