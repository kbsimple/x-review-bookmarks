# Phase 9 — UI Design Contract

## Design System

| Property | Value |
|----------|-------|

## Spacing Scale

Declared values (Tailwind defaults, multiples of 4):

| Token | Tailwind | Value | Usage |
|-------|----------|-------|-------|
| xs | p-1, gap-1 | 4px | Icon gaps, tight inline |
| sm | p-2, gap-2, mb-2 | 8px | Compact element spacing |
| md | p-3, gap-3, mb-3 | 12px | Small section gaps |
| base | p-4, gap-4, mb-4 | 16px | Default element spacing, card padding |

**Exceptions:** None — all spacing uses Tailwind scale

- "View on X" links on post cards

- Navigation active states

## Implementation Notes

1. **Jinja2 Macros:** Create `_post_card.html` and `_embedded_post.html` in `templates/components/` for DRY rendering
2. **Repository JOIN:** Modify `PostsRepository.get_paginated()` to LEFT JOIN `embedded_posts` table
3. **Template Variables:** Pass `post.embedded_post` to macro for retweets/quotes
4. **Null Safety:** Always check `{% if post.embedded_post %}` before accessing embedded properties
5. **Loading States:** Consider skeleton placeholder while fetching (Claude's Discretion area)
6. **Error Handling:** Handle failed embedded post fetches gracefully (Claude's Discretion area)
