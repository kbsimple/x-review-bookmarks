# Milestone v1.7: Deep Linking

**Status:** ✅ SHIPPED 2026-06-14
**Phases:** 17
**Total Plans:** 2

## Overview

Add shareable deep link URLs to the static viewer. A share icon (📤) on each post card copies a `#post-{id}` hash URL to the clipboard. Opening that URL opens the viewer in a focused carousel view (filters cleared, that post shown) with an "XBM Home" button in the header to return to the full viewer.
